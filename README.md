# Game Of Life

A shell version of the famous **Game Of Life** by John Conway entirely written in Elixir.

```
---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---
                        ||| |||                         ||| |||                                

                                                        |||     |||                     |||    

                |||     |||                                                     ||| |||     |||

                                                                                ||| |||     |||

                        |||                 |||                         |||         ||| |||    

        |||             |||             |||     |||                         |||                

                                            |||                                                

                    ||| ||| |||             |||                                                

|||     |||                 ||| ||| |||     ||| |||                                            

||| |||             |||             |||         |||                                            

                    ||| ||| ||| |||             |||                                            

                        ||| ||| ||| ||| |||                                                    
---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+---

w: 24, h: 12, iterations: 8
```


## Starting The Game

To start the game run the mix task:  `$ mix start`
